## Overview

This chatbot utilizes GPT-3.5 Turbo and the DALLÂ·E 3 API to generate a movie plot, title, and poster based on prompts entered by users. The movie generator is finely tuned to excel at creating engaging, professional movies. Each movie synopsis is approximately 350 words long.

## Features

-   Type in a movie idea.
-   Generate a movie story based on the user input.
-   Generate a movie title.
-   Generate a movie poster.
